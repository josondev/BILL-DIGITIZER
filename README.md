# NVIDIA NIM OCR & NL2SQL Pipeline

Invoice OCR and Natural Language Query System using NVIDIA NIM APIs.

## Features
- Extract data from invoice images using NVIDIA Vision models
- Query invoice data using natural language
- SQLite database storage
- Interactive Streamlit UI

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set NVIDIA_API_KEY environment variable
4. Run: `streamlit run backend/ui.py`

## Deployment
Deployed on Streamlit Cloud with NVIDIA NIM integration.
